Theoretical background
Core concepts behind CFG simplification — essential for formal languages, automata theory, and compiler design.
◆
Context-free grammar
A 4-tuple G = (V, Σ, R, S) where V = non-terminals, Σ = terminals, R = production rules, S = start symbol. Generates a context-free language (CFL).
⊘
Useless symbols
A symbol is useless if it is either non-generating (cannot derive any terminal string) or unreachable (no sentential form containing it exists from S).
𝜖
Nullable variables
A variable A is nullable if A ⇒* ε. These must be identified before eliminating ε-productions to preserve the generated language.
→
Unit productions
A rule of the form A → B where A and B are both non-terminals. These create unnecessary chains that can be shortcircuited via transitive closure.
●
Generating symbols
A variable A is generating if A ⇒* w for some terminal string w ∈ Σ*. Computed bottom-up: terminals are trivially generating, propagate upward.
◈
Reachable symbols
A symbol X is reachable if S ⇒* αXβ for some strings α, β. Computed via BFS/DFS from the start symbol through all productions.
Step 1 — Removing useless symbols (algorithm)
1a
Find generating set: Initialize Gen = {all terminals}. Repeatedly add variable A to Gen if ∃ rule A → α where all symbols of α are in Gen. Repeat until no change.
1b
Remove non-generating: Delete all variables not in Gen, and all rules containing them.
1c
Find reachable set: Initialize Reach = {S}. For each variable in Reach, add all symbols appearing in its rule bodies. Repeat until stable.
1d
Remove unreachable: Delete all variables and terminals not in Reach, along with their rules.
Step 2 — Eliminating ε-productions (algorithm)
2a
Find nullable set: Initialize Null = {A | A → ε ∈ R}. Then repeatedly add A if ∃ A → B₁B₂…Bₖ where every Bᵢ ∈ Null. Repeat until stable.
2b
Rewrite rules: For each rule A → α, generate all versions of α obtained by optionally omitting each nullable symbol in α (excluding the all-omit case which produces ε, unless A is the start).
2c
Remove ε rules: Delete all rules of the form A → ε (except S → ε if ε was in the original language).
Step 3 — Removing unit productions (algorithm)
3a
Compute unit pairs: For each variable A, find all B such that A ⇒* B using only unit productions (A = B is included). This is the transitive closure of the unit relation.
3b
Build new rule set: For each unit pair (A, B) and each non-unit rule B → α, add A → α to the new grammar.
3c
Remove unit rules: Delete all original unit productions A → B. Keep only the new expanded rule set.
Language equivalence
Theorem: Let G be a CFG. Then there exists a CFG G' with no useless symbols, no ε-productions (except S → ε if ε ∈ L(G)), and no unit productions such that L(G') = L(G) \ {ε} (or L(G') = L(G) if ε ∉ L(G)).

Each simplification step is a conservative transformation — it only removes redundant or problematic rules while introducing equivalent alternatives. The proof uses induction on derivation length to show that every string in L(G) has a corresponding derivation in G' and vice versa.
